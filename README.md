# Object-Oriented Programming Assessment - Car Park System

This is a simplied car park system using Object-Oriented Programming(OOP) concepts for assessment project.

The system will consist of a car park, sensors, and displays classes to track the cars entering and exiting and the availability of parking bays.


## Tips to run unit tests
Update import as below:

If run test_car_park, update car_park.py(from src.sensor import Sensor, from src.display import Display) to solve no module found issues.

Code to run unit tests: python -m unittest tests.test_car_park


## Tips to run main.py
In the root folder, git bash run: python src/main.py
